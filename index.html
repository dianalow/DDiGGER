DDiGGER

Data-Driven Grouping for Gene Expression in R

DDiGGER is a data-driven grouping method for significant gene and gene pair selection.

Features

  • Uses information from microarray intensities and patient survival information
  • Identifies survival significant genes (1D) and gene pairs (2D)
  • Produces list of genes via Cox proportional hazards model and corresponding statistics
  • Generates survival plots for chosen genes

Requirements

This software package requires :

  • R version 2.14.0 and above
  • R packages : survival, tkrplot. To install, type 
    install.packages("package-name")
    at R command prompt.
  • tcltk (pre-installed in most Linux distros; comes with Windows installation of R; visit CRAN for Mac instructions)
  • Tktable (tcltk extension) for Linux. The Tktable package is a standard Linux/Unix source package so it can be unzipped with tar zxvf and installed with configure; make; make install assuming that you have the appropriate tcl-devel and tk-devel packages installed.
  • GSView to view postscript plots
